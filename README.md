This repository contains a containerized Guestbook web application
designed to demonstrate Kubernetes orchestration concepts such as:

- Docker multi-stage builds
- Kubernetes Deployments and Services
- Horizontal Pod Autoscaling (HPA)
- Rolling updates and rollbacks

The project is organized as a portfolio-ready implementation based on
a real-world orchestration scenario.

## Repository Structure

- src/ Application source code (v1 and v2)
- docker/ Dockerfiles for container images
- k8s/ Kubernetes manifests (deployments, service, HPA)
- docs/ Technical documentation
- evidence/ Execution and validation screenshots


## Technologies Used

- Docker
- Kubernetes
- Horizontal Pod Autoscaler (HPA)
- HTML / JavaScript frontend

## Notes

This repository focuses on architecture, orchestration, and deployment
practices rather than local execution.
