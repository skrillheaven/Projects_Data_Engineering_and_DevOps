# ETL Pipeline: Largest Banks Market Capitalization Dataset

## Project Overview

This project implements an automated **ETL pipeline in Python** that extracts market capitalization data of the world’s largest banks from a public web source, transforms the dataset into multiple currencies using exchange rate information, and loads the processed data into structured storage formats including **CSV files** and a **SQLite database**.

The pipeline also executes SQL queries and records execution progress through structured logging, simulating a real-world recurring financial reporting workflow.

---

## Project Objectives

- Extract tabular financial data from a web archive source
- Transform market capitalization values into GBP, EUR, and INR
- Store processed datasets in CSV format
- Persist structured data into a SQLite database table
- Execute analytical SQL queries on stored data
- Implement structured logging across ETL pipeline stages


---

## Execution Workflow

Pipeline execution sequence:

1. Extract banking dataset from archived webpage
2. Transform market capitalization values into GBP, EUR, and INR
3. Save transformed dataset into CSV format
4. Store dataset into SQLite database table
5. Execute analytical SQL queries
6. Log execution progress into structured log file

## Database Queries Implemented

Example queries executed during pipeline validation:

Retrieve full dataset:

```
SELECT * FROM Largest_banks
```

Average market capitalization in GBP:

```
SELECT AVG(MC_GBP_Billion) FROM Largest_banks
```

Retrieve top 5 banks:

```
SELECT Name FROM Largest_banks LIMIT 5
```
