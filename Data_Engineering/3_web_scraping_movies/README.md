# Web Scraping Movies Dataset

This project extracts the top ranked films from a public webpage using Python.

## Technologies
- Python
- BeautifulSoup
- Requests
- Pandas
- SQLite

## Pipeline

1. Extract movie data from HTML page
2. Transform into structured dataframe
3. Load results into:
   - CSV dataset
   - SQLite database

## Files

webscraping_movies.py -> main scraping script  
top_50_films.csv -> dataset output  
Movies.db -> SQLite database
