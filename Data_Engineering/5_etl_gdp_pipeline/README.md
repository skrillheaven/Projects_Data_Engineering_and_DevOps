# ETL Pipeline – GDP Data Processing

This practice project demonstrates the implementation of a complete
ETL (Extract, Transform, Load) workflow using Python.

The objective of the pipeline is to extract GDP data from structured
sources, transform and clean the dataset, and export the processed
results into a structured output file for downstream analysis.

---

## Project Objectives

The pipeline performs the following tasks:

- Extract GDP data from source files
- Transform and standardize dataset values
- Convert GDP values into millions of USD
- Log execution steps for traceability
- Export transformed dataset into CSV format


Loads raw GDP dataset from source file.

### Transform

Standardizes values and converts GDP figures into millions USD.

### Load

Exports transformed dataset into CSV output file.
